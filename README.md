# vue-weather

> A weather demo app using [Vue.js 2](http://vuejs.org/guide/) & [Vuex](https://github.com/vuejs/vuex). Weather information is provided in real time using Weatherbit API.





To start:

```bash
$ npm install
```

To develop:

```bash
$ npm run serve
```
